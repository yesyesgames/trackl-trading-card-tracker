# Trackl ‚Äì Trading Card Tracker

Trackl is a modern, minimalistic app for trading card game (TCG) collectors.
Easily manage your collection, organize variants, and add custom attributes like language, condition, or special coatings.

<img src="http://misc.yesyesgames.de/trackl/feature-graphic.webp?" width="*"/>

### üß© Key Features
- Smart search by card name or ID
  - Type letters to search by name 'name' or 'name xx'
  - Use six numbers like `123/123` or just `123123` to search by id
  - Pair up to three letters and three numbers to search sets : `set/123` or `set123`
- Track multiple variants* of the same card
  - Supports various types of foils and illustration labels
- Add cosmetic attributes like language, condition or coating
- Organize and filter your collection easily
- View market price** information
- Clean, distraction-free design
- And more

_*Variants and prices are sourced from TCGPlayer. If no price data is available for a variant, it may not be shown._  
_**Price reflects average market values only and does not represent individual listings._
